## 🚀 Getting Started

### Prerequisites

```bash
node >= 18.0.0
npm >= 9.0.0
python >= 3.9
mongodb >= 6.0
```

### 1️⃣ Clone Repository

```bash
git clone https://github.com/SatyaYellapu/iap-mg-genai.git
cd iap-mg-genai
```

### 2️⃣ Backend Setup

```bash
cd backend
npm install
cp .env.example .env
# Fill in your .env values (see Environment Setup below)
npm run dev
# Server starts at http://localhost:5000
```

### 3️⃣ Frontend Setup

```bash
cd frontend
npm install
npm run dev
# App starts at http://localhost:3000
```

### 4️⃣ Python Services Setup

```bash
cd python-services

# PDF Service
cd pdf_service
pip install -r requirements.txt
python app.py   # Port 5001

# SMS Service (new terminal)
cd sms_service
pip install -r requirements.txt
python app.py   # Port 5002

# Translation Service (new terminal)
cd translation_service
pip install -r requirements.txt
python app.py   # Port 5003
```

### ⚡ Quick Start (Windows — all services at once)

```bat
# Double-click this file:
start-all.bat
```

## 🔐 Environment Setup

Create `backend/.env` with the following:

```env
# ── Server ────────────────────────────────────────
PORT=5000
NODE_ENV=development
JWT_SECRET=your_super_secret_jwt_key_min_32_chars

# ── Database ──────────────────────────────────────
MONGODB_URI=mongodb://localhost:27017/medical-diagnosis

# ── Google Gemini AI (Required) ───────────────────
GEMINI_API_KEY=your_gemini_api_key
# Get from: https://makersuite.google.com/app/apikey

# ── Email — Brevo SMTP ────────────────────────────
EMAIL_USER=your_brevo_smtp_user
EMAIL_PASSWORD=your_brevo_smtp_password
EMAIL_HOST=smtp-relay.brevo.com
EMAIL_PORT=587

# ── Twilio SMS ────────────────────────────────────
TWILIO_ACCOUNT_SID=your_twilio_account_sid
TWILIO_AUTH_TOKEN=your_twilio_auth_token
TWILIO_PHONE_NUMBER=+1xxxxxxxxxx

# ── Python Services ───────────────────────────────
PDF_SERVICE_URL=http://localhost:5001
SMS_SERVICE_URL=http://localhost:5002
TRANSLATION_SERVICE_URL=http://localhost:5003

# ── Frontend ──────────────────────────────────────
FRONTEND_URL=http://localhost:3000
ALLOWED_ORIGINS=http://localhost:3000
```

> ⚠️ **Important**: Use Brevo SMTP (not Gmail) for email. Gmail blocks app login by default.

## 🔌 API Reference

<details>
<summary><b>🔐 Authentication</b></summary>

| Method | Endpoint | Description |
|--------|----------|-------------|
| `POST` | `/api/auth/register` | Register new user |
| `POST` | `/api/auth/login` | Login & get JWT token |
| `GET` | `/api/auth/me` | Get current user profile |

</details>

<details>
<summary><b>🩺 Diagnosis</b></summary>

| Method | Endpoint | Description |
|--------|----------|-------------|
| `POST` | `/api/diagnosis/submit-and-analyze` | Submit symptoms + get AI diagnosis |
| `GET` | `/api/diagnosis/sessions` | Get all diagnosis sessions |
| `GET` | `/api/diagnosis/session/:id` | Get specific session details |

</details>

<details>
<summary><b>📄 PDF & Communication</b></summary>

| Method | Endpoint | Description |
|--------|----------|-------------|
| `POST` | `/api/pdf/generate` | Generate PDF report (blob) |
| `POST` | `/api/email/send-report` | Email report to user |
| `POST` | `/api/sms/setup-reminders` | Schedule SMS reminders |
| `POST` | `/api/sms/test` | Send test SMS |

</details>
